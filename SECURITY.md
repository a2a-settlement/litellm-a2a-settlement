# Security Policy

## Supported Versions

We release patches for security vulnerabilities for the current major version.

## Reporting a Vulnerability

Please report security vulnerabilities via [GitHub Security Advisories](https://github.com/a2a-settlement/litellm-a2a-settlement/security/advisories/new). Do not open a public issue.

We aim to acknowledge within 48 hours and provide an initial assessment within 7 days.
